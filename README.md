# Java String Operations

This repository contains basic Java programs that demonstrate common **String operations**.
These programs are useful for beginners who are learning Java and practicing string manipulation.

## 📌 Programs Included

### 1. String Concatenation

This program takes the **first name** and **last name** as input from the user and combines them into a **full name** using string concatenation.

**Example Output**

Enter the First Name: Hetal
Enter the Last Name: Bagal
The FullName is: Hetal Bagal


### 2. String Comparison

This program compares two strings entered by the user using the `equals()` method.

**Example Output**

Enter the First Name: Java
Enter the Last Name: Java
They are the same Strings

## 🛠 Technologies Used

* Java
* Scanner class for user input


## 🎯 Purpose

The purpose of this repository is to practice **basic Java String operations** and improve programming skills.
